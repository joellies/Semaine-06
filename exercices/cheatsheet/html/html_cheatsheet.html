Cheat Sheet : HTML5

Structure de base d'une page web

Pour commencer

Élément Description
!DOCTYPE html indiquer au navigateur quelle version d'HTML est utilisée sur la page
html lang="en" balise principale, prend l'attribut langue de la page
head en-tête du document contenant les métadonnées
meta charset="UTF-8" balise méta pour les caractères utilisés
meta name="viewport" content="width=device-width, initial-scale=1.0" zône visible de la page pour l'user
link rel="stylesheet" href="style.css" type="text/css" relier la page CSS à l'html
title titre de la page qui apparait dans l'onglet du navigateur
body corps de la page
