Cheat Sheet : Javascript

Les bases

Où placer le code Javascript

  • Si le document script.js est une feuille à part, on va dans le document HTML et on place la balise script src="" entre la balise fermante du body et celle de l'html.
  • Si on veut mettre le script dans l'html, on met une balise script type ="text/javascript" dans le document html.

Comments

  • //
    commentaire d'une ligne 
  • /**/
    commentaire de plusieurs lignes

Output

  • console.log()
    afficher dans la console du navigateur 
  • alert()
    afficher une box alert, empêcher d'aller plus loin 
  • confirm()
    afficher une boîte de dialogue (permet de construire une condition, oui/non avec true/false) 
  • prompt()
    afficher une boîte de dialogue (permet de mettre une fonction à l'intérieur)

Delay

setTimeOut(function(){}, 1000);
délai d'une seconde (1000 milisecondes) avant d'enclencher une action

DOM

Attributs

  • .name
    retourne le nom de l'élément
  • .value
    retourne la valeur de l'élément
  • .lenght
    retourne la longueur de l'élément (tableau par exemple)

Méthodes

  • document.addEventListener
    attache un "écouteur" d'événement au document (à un bouton par exemple)
  • document.body
    retourne le body du document HTML
  • document.createElement("")
    crée un nouveau élément
  • document.createTextNode("")
    crée un nouveau "noeud" de texte
  • document.getElementById("")
    retourne l'élément grâce à son "id"

Les variables

Le nom d'une variable est unique et claire. On utilise le camelCase pour la définir. Le = assigne la variable à une valeur : une chaîne de caractères (""), un nombre, un booléen (true/false), un array (tableau), un objet (avec des propriétés et des valeurs). 

  • const myName = "Joéllie"

    • déclarer une variable qui ne contiendra qu'une valeur qui ne pourra être accessible seulement si elle précédemment déclarée.

  • let myAge = 29

    • déclarer une variable, mais sans la contrainte d'assignation unique.
