Cheat Sheet : Javascript

Les bases

Où placer le code Javascript

  • Si le document script.js est une feuille à part, on va dans le document HTML et on place la balise script src="" entre la balise fermante du body et celle de l'html.
  • Si on veut mettre le script dans l'html, on met une balise script type ="text/javascript" dans le document html.

Comments

  • // : commentaire d'une ligne
  • /**/ : commentaire de plusieurs lignes

Output

  • console.log() : afficher dans la console du navigateur
  • alert() : afficher une box alert, empêcher d'aller plus loin
  • confirm() : afficher une boîte de dialogue (permet de construire une condition, oui/non avec true/false)
  • prompt() : afficher une boîte de dialogue (permet de mettre une fonction à l'intérieur)
